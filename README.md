# mlops chalenge



## Usage
To download dataset and cleaning:

```
make cleaning_process
```

To training yout KNN dataset:

```
make training_process
```

To launch movie_recommender endpoint:

```
make build_endpoint
```

To build all:
```
make build
```

To execute unit_test:
```
make unit_test
```
